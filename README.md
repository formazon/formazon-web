# Formazon.com ‚Äî Personal Product & Design Portfolio

This repository contains the source code for **formazon.com**,  
the personal website and portfolio of **Farid Rafikov** ‚Äî a product founder and designer working across mobile apps, AI platforms, and robotics-driven systems.

The site showcases projects such as **TRA Robotics**, **Jungle**, **AppForType**, **Fuelet**, **Explyt**, and **Esprito**.  
It is built with **Next.js**, **React**, and **Tailwind CSS**, and deployed on **Vercel**.

---

## Getting Started

First, run the development server:

```bash
npm run dev
```
